Rss::Blocks is a codeblocks plugin that shows rss feeds from the channel of your choice in a window that can easily be customized using html.

Doumentation

You can read the complete documentation for Rss::Blocks online that explains its usage and internals right here

Download

You can download Rss::Blocks from the file manager at sourceforge. There you should find a zip file containing the source code, datafiles and the Code::Blocks project to build it.

Download Now!

Installing

  • Download the latest release
  • Build Rss::Blocks with Code::Blocks
  • Copy default.rbc and default.rbi to the share/CodeBlocks/ folder
  • Copy rssblocks.png and rssblocks-off.png to the share/CodeBlocks/images/settings/ folder
  • Then install rssblocks.cbplugin with "Manage Plugins"
